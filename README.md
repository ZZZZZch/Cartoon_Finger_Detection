# Cartoon_Finger_Detection

## 先安装git-lfs
```shell
curl -s https://packagecloud.io/install/repositories/github/git-lfs/script.rpm.sh | bash -

yum makecache

yum install -y git-lfs
```

## 克隆代码

```shell
git lfs clone https://github.com/ZZZZZch/Cartoon_Finger_Detection.git
```

## 直接运行  set_up.sh
```shell
sh set_up.sh
```
